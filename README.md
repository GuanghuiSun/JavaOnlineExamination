# JavaOnlineExamination
Java在线考试系统,是基于swing界面的Client/Server结构的在线考试系统
## Java课设
基于TCP网络通讯实现在线考试功能：

（1）教师端：

①教师登录后，将学生信息导入到数据库中

②设置试卷结构，包括有哪些题型，每种题型的数量等

③批阅主观题，给出成绩

④可以查看所有学生的作答情况及成绩

（2）学生端：

①考试时，首先登录系统，连接服务器

②服务器按照试卷结构要求，随机抽选一套题，发送到客户端

③用户将作答结果上传给服务器端

④系统自动批阅客观题，给出客观题得分

⑤可以查询批阅结果及自己的成绩

（3）系统：

①自动计算每位同学的总分
